📄 Street View Publish API Reference
Complete 4-step workflow. EXIF extracted automatically on upload.
Publish Flow
1. POST /photo:startUpload → uploadUrl
2. PUT raw image bytes to upload URL
3. POST /photo with full pose payload
4. PATCH connections after IDs are known
